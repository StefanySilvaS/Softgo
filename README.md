# 🌱 SoftGo

**SoftGo** é uma aplicação web desenvolvida em React para ajudar pessoas a registrar vontades e acompanhar experiências de forma simples e acolhedora.

O projeto foi pensado com foco em experiência do usuário (UX), priorizando clareza, organização e conforto durante o uso.

---

## Funcionalidades

-  Registro de vontades por categorias  
-  Entendimento de como cada experiência funciona  
-  Orientações de preparo e segurança  
-  Check final da experiência  
-  Histórico pessoal com datas e anotações  

---

## Tecnologias Utilizadas

- React
- JavaScript
- HTML5
- CSS3
